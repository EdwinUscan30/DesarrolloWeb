Paso 1° | Aprender HTML

El HTML es parte importante de las reglas web, en general con otras tecnologías como el CSS o el JavaScript. Debido a que este lenguaje es la base para edificar páginas en Internet, es fundamental conocerlo para generar nuestras webs.

¿Qué es y para qué sirve el HTML?
HTML son las siglas en inglés para Hypertext Markup Language, que se traduce como lenguaje de marcación de hipertexto. Pese a tener un nombre difícil, el HTML no es más que un lenguaje utilizado para generar páginas webs mediante marcadores (tags) y atributos, que definen cómo el contenido será presentado en un navegador web.
